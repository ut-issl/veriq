# Installation

## Requirements

- Python 3.14 or later

## Using pip

```bash
pip install veriq
```

## Using uv (recommended)

If you're using [uv](https://docs.astral.sh/uv/), you can add veriq to your project:

```bash
uv add veriq
```

## Verify Installation

Check that veriq is installed correctly:

```bash
veriq --version
```

You should see output like:

```
veriq 0.2.0
```

## Next Steps

Now that you have veriq installed, proceed to the [tutorial](tutorial/index.md) to learn how to use it.
